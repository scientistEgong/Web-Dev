## Reflection
### What I Learned
While working on these projects, I gained practical experience in several core areas of web development, which include:
- Structuring web pages using **HTML elements and Semantic tags**
- Understanding the importance of **Proper file organization and linking pages**
- Deploying a website Online using **Vercel**
  
These exercises helped reinforce my understanding of how front-end web development technologies work together to create a functional and accessible web page.

---

### Challenges Faced
Some of the challenges I encountered during the development process includse:
- Correctly structuring HTML elements to maintain proper page heirarchy
- Styling pages consistently using CSS
- Debugging errors such as broken links or incorrect tag usage
- Ensuring that pages are displayed correctly across different browsers
- Understanding the deployment process and correctly publishing the project online.
  
Overcoming these challenges improved my problem-solving skills and deepened my understanding of web development fundamentals.
